Hack Developer Experience in VS Code

Hack is a programming language for the HipHop Virtual Machine (HHVM), created by Facebook
as a dialect of PHP.The language implementation is open-source, licensed under the MIT License.
Hack allows programmers to use both dynamic typing and static typing. This kind of a type system
is called gradual typing, which is also implemented in other programming languages such as ActionScript.
Hack's type system allows types to be specified for function arguments, function return values, and class properties;
however, types of local variables are always inferred and cannot be specified.

Using Visual Studio Code with the VSCode-Hack extension, provides standard IDE-like
features such as syntax highlighting, go-to-definition, and in-line display of Hack errors.
